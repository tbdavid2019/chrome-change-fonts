# Privacy Policy / 隱私政策

Last updated: 2026-03-31

## 繁體中文

### Font Changer 改字體 333 隱私政策

`Font Changer 改字體 333` 尊重您的隱私。本擴充套件的設計目的是讓使用者可以在瀏覽網站時套用自己選擇的字體，並盡量以最少資料運作。

### 1. 我們會收集哪些資料

本擴充套件不會蒐集、儲存或傳送您的個人身份資訊，例如姓名、電子郵件、電話、地址或付款資訊。

本擴充套件只會在您的瀏覽器內使用以下資料：

- 您選擇的字體名稱
- 您是否啟用字體替換功能

上述資料會儲存在 Chrome 提供的 `chrome.storage.sync` 中，用來在不同裝置或同一瀏覽器工作階段中保留您的設定。

### 2. 權限用途說明

本擴充套件使用以下權限：

- `fontSettings`
  用於讀取您系統中可用的字體清單，讓您可以在 popup 中選擇字體。
- `storage`
  用於儲存您的字體設定與啟用狀態。
- `activeTab`
  用於將您選擇的字體樣式套用到目前作用中的分頁。

### 3. 我們如何使用資料

本擴充套件只將上述資料用於以下目的：

- 顯示可選擇的系統字體
- 套用您選擇的字體到目前網頁
- 保留您的設定，避免每次重新選擇

### 4. 資料傳輸與第三方服務

本擴充套件不會將您的資料傳送到開發者伺服器，也不會出售、出租或分享您的資料給第三方。

本擴充套件不使用分析工具、不使用廣告追蹤、不建立使用者行為檔案。

### 5. 網頁內容存取說明

本擴充套件會在您開啟的網頁中注入樣式，以改變頁面文字顯示使用的字體。這項操作只發生在您的本地瀏覽器中，目的是修改顯示效果，不會將網頁內容上傳或傳送到外部服務。

### 6. 資料安全

本擴充套件不維護外部資料庫，也不在開發者端保存任何使用者資料。所有設定僅依賴瀏覽器提供的儲存機制。

### 7. 兒童隱私

本擴充套件並非以兒童為目標對象，也不會刻意蒐集兒童的個人資料。

### 8. 政策更新

若本隱私政策未來有變更，我們會在本檔案中更新內容與日期。

### 9. 聯絡方式

若您對本隱私政策有任何疑問，請透過本專案的 GitHub repository 與開發者聯絡。

## English

### Privacy Policy for Font Changer 改字體 333

`Font Changer 改字體 333` respects your privacy. This extension is designed to let users apply their preferred fonts while browsing websites, and it operates using the minimum data required.

### 1. Information We Collect

This extension does not collect, store, or transmit personally identifiable information such as your name, email address, phone number, home address, or payment details.

The extension only uses the following information locally in your browser:

- The font name you selected
- Whether font replacement is enabled

This information is stored in Chrome's `chrome.storage.sync` so your preferences can persist across browser sessions and supported synced environments.

### 2. Why the Extension Requests Permissions

This extension uses the following permissions:

- `fontSettings`
  Used to read the list of fonts available on your system so you can choose one in the popup.
- `storage`
  Used to save your selected font and enabled state.
- `activeTab`
  Used to apply your selected font style to the currently active tab.

### 3. How We Use the Information

The extension uses this information only to:

- Show the available system fonts
- Apply your selected font to the current webpage
- Preserve your preference so you do not need to set it again each time

### 4. Data Sharing and Third-Party Services

This extension does not send your data to developer-operated servers and does not sell, rent, or share your data with third parties.

This extension does not use analytics, advertising trackers, or profiling tools.

### 5. Webpage Access

This extension injects style rules into webpages you open in order to change the font used for visible text. This process happens locally in your browser for display purposes only. Webpage content is not uploaded or transmitted to external services.

### 6. Data Security

This extension does not maintain an external database and does not store user data on developer-controlled systems. Preferences rely only on browser-provided storage.

### 7. Children's Privacy

This extension is not directed to children and does not knowingly collect personal information from children.

### 8. Changes to This Policy

If this privacy policy is updated in the future, the revised content and date will be updated in this file.

### 9. Contact

If you have any questions about this privacy policy, please contact the developer through this project's GitHub repository.
